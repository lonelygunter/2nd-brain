###### @ [Greedy search] Pseudocodice:
```python
OPEN = { (start node, h(start node)) }
CLOSED = empty;
while (OPEN != empty):
	X = extract the state with the "lowest value of h(X)" from OPEN
	if (X = goal state):
		return success # h=0
	SUCCESSORS = Successor function (X)
	CLOSED.put(X)
	discard any successors contained in OPEN or CLOSED
	OPEN = SUCCESSORS
```
